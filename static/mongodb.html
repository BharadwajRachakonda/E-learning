Introduction to MongoDB

MongoDB is a NoSQL database that stores data in JSON-like documents called BSON. It is highly scalable and flexible for modern application development. 

npm install mongodb

Basic Commands

Insert Document

db.collection.insertOne({ name: "Alice", age: 25 });

Find Documents

db.collection.find({ age: { $gt: 20 } });

Comparison Operators

  • $gt: Greater than
  • $lt: Less than
  • $gte: Greater than or equal to
  • $lte: Less than or equal to 
db.collection.find({ qty: { $gte: 5 } });

Regex and Pattern Matching

Search for documents that match a specific pattern using $regex. 

db.collection.find({ name: { $regex: /^A/ } });
